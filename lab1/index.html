Programação Web




Na Unidade Currricular de Programação Web abordaremos os principios básicos associados ao desenvolvimento para a web, tanto do lado back-end como front-end.

Aprender-se-ão as segiuntes tecnologias 🌐:


    HTML

    CSS

    Python

    Django

    JavaScript

De forma a consolidar todos os conhecimentos, será feito um laboratório semanal, de entrega obrigatória, de aplicação da matéria lecionada. Irá ser desenvolvido também um projeto.
